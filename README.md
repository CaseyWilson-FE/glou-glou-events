# Glou Glou Private Events

Landing page for events.glouglou.nz

## Structure
- `index.html` — the page
- `Images/` — all photos
- `fonts/` — Alti Regular and Galvji

## Deployment
Hosted on Netlify, connected to this repo. Any commit to main auto-deploys.

## To update
Edit index.html, commit, and Netlify deploys automatically.
